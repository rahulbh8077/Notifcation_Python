# 🔔 Desktop Notification App

A lightweight desktop notification application built with **Python** using the **Plyer** library. This project demonstrates how to display native system notifications through a simple and beginner-friendly implementation.

---

## 📖 Overview

The Desktop Notification App allows users to generate desktop notifications directly from a Python script. It is designed as a simple project to explore desktop automation and understand how Python can interact with the operating system's notification service.

## ✨ Features

- Display native desktop notifications
- Lightweight and easy to use
- Simple and clean Python implementation
- Beginner-friendly project structure
- Easy to customize notification title, message, icon, and duration

## 🛠️ Tech Stack

- Python 3
- Plyer

## 🚀 Getting Started

### Prerequisites

- Python 3.x installed

### Installation

```bash
git clone https://github.com/your-username/Desktop-Notification-App.git
cd Desktop-Notification-App
pip install -r requirements.txt
```

Or install manually:

```bash
pip install plyer
```

## ▶️ Usage

```bash
python notification.py
```

A desktop notification will appear with the configured title and message.
